# AQN - Asked Questions Notes
**AQN** is a software tool designed for creating, managing, and utilizing notes derived from lecture materials, primarily for university courses. It facilitates the generation of questions and answers for each topic and provides a graphical visualization of the content, supporting both hierarchical directed tree structures and undirected simple graphs.
